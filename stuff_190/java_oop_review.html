Advanced Data Types

Wrapper Classes (Integer, Double)

In Java, primitives (like int and double) are efficient but aren't objects. Wrapper classes allow primitives to be treated as objects.

  • Integer: The wrapper class for int.
  • Double: The wrapper class for double.
  • Note: These are Immutable. Once created, the value cannot change.
BigInteger & BigDecimal

Sometimes standard primitives aren't enough.

  • BigInteger: Used for integers that are too large to fit in a long (64-bit). It can store arbitrarily large integers.
  • BigDecimal: Used for precise decimal calculations. Floating-point types (float, double) can have rounding errors. BigDecimal is essential for financial calculations (money).
